CONCLUSION

In this module, you have learned how to assess a firm’s profitability and value creation potential. You now know that the return on equity is driven by a firm’s margin and its asset turnover, which together determine its return on assets. This, compared with the average cost of debt, determines the effect of the use of financial leverage on the return for the shareholders.

You now understand that, in order to create value for shareholders, the return on assets should exceed the weighted average cost of capital, which represents the average cost of all funding sources used by the firm. Equivalently, the return on equity should exceed the firm’s cost of equity, which is higher the more risky the firm is. Understanding profitability is hence key in understanding value creation.
