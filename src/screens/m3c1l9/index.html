AQUASAIL’S TOTAL DEPRECIATION

Depreciation is an additional expense that has to be taken into account in determining Aquasail’s first quarter profit. Aquasail used its buildings and machinery for its operations and their value has decreased due to this usage.

Select each item to learn more.

BUILDINGS AND MACHINERY

The estimated life time of the buildings is 20 years (or 80 quarters), while that of the machinery is 4 years (or 16 quarters). After that, it is expected that these assets will not have any value any more.

Following the linear depreciation method, we estimate that the decrease in value of the assets occurs linearly over time.

The quarterly decrease in value of the buildings – or the amount that has to be depreciated – equals the buildings’ initial acquisition value, or 8000 total value divided by 80 quarters = 100.

The quarterly depreciation of the machinery equals 4800 total value divided by 16 quarters = 300.

Aquasail’s total depreciation hence amounts to 100 (buildings) + 300 (machinery) = 400.

THE LAND

Note that the land is not depreciated: from an accounting point of view, it is assumed that land keeps its value; it doesn’t lose value by using it. Put differently, land has an infinite expected lifetime.
